# Repository contents

- `tree_coarsening/` — package source.
- `tests/` — schema-1 correctness, conformance, parity, and regression tests.
- `benchmarks/` — release performance notebooks and shared benchmark helpers.
- `notebooks/` — small syntax/example notebooks for individual coarseners.
- `experiments/` — project-specific CARBANAK notebooks requiring local data.
- `docs/` — supplementary design notes and a copy of the API contract.
- `audit/` — release/audit evidence and protected BPE baseline files used by tests.
- `CURRENT_SCHEMA1_API.md` — normative schema-1 contract.
- `CONTRIBUTING_COARSENERS.md` — contributor guide for new coarseners.
- `CONTRACT_TRACEABILITY.md` — contract-to-code/test map.
